## MacroSynthetix Terminal

Asset managers face a constant stream of market-moving information including macroeconomic releases, central bank commentary, geopolitical developments, and regulatory announcements. This information is often fragmented across multiple sources, making it difficult for analysts to track evolving themes and assess cross-asset risks in real time.

**MacroSynthetix** is an AI-assisted macro research terminal that transforms unstructured financial news into structured macro insights. The system ingests live news feeds and maps events to macroeconomic themes such as inflation, growth, geopolitics, and central bank policy.

To track how themes evolve, the terminal maintains an **institutional memory** using TF-IDF retrieval and cosine similarity, allowing it to surface historically similar macro events. A **causal macro knowledge graph** then propagates shocks across assets using graph traversal, identifying second-order effects across rates, equities, FX, and commodities.

An interactive dashboard allows users to explore news developments, track “hot” macro themes, and review historical precedents. The system also generates **risk implications** for each event by mapping shocks to potential cross-asset volatility and portfolio exposures.

MacroSynthetix aims to reduce information overload by combining AI, retrieval methods, and deterministic macro models to support faster macro analysis and decision-making.
